# gubtool

A tui practice tool for soulsborne games. Supports only elden ring at the moment, I will work on ds2 next.

Works only on linux and has been tested on arch, fedora, and gentoo. KDE konsole does not properly display the dimming effect on inactive lists, so consider using another terminal emulator.

Thanks to Shilkey and Recycle for being very helpful chunguses, and for making [TarnishedTool](https://github.com/borgCode/TarnishedTool), where much of the logic and values is taken from.
